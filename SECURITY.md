# Security Policy — MR NOTHING

## Reporting

Do NOT open a public issue for security vulnerabilities.

Report privately via [GitHub Security Advisories](../../security/advisories/new). We respond within 72 hours.

## Scope

- Android agent runtime (command execution, permissions)
- Backend API authentication
- Memory store (user data privacy)
- Voice processing pipeline
- SDK security

## Priority Areas

- Arbitrary command execution through agent interface
- User data exposure via memory store
- API authentication bypass
- Voice data interception or unauthorized recording
